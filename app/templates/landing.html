Frequently asked questions

Everything you need to know about getting started with {{ app_name }}.

{{ app_name }} supports full-scope penetration testing including network reconnaissance, vulnerability scanning, web application testing, service enumeration, and automated exploitation — all orchestrated by AI agents within sandboxed containers.

All tools run in isolated, sandboxed containers with configurable scope limits and rules of engagement. The platform enforces target boundaries and supports human-in-the-loop approval for high-risk actions. We recommend testing against staging environments first.

{{ app_name }} is model-agnostic and supports multiple LLM providers including OpenAI, Anthropic, and local models via Ollama. You can configure your preferred provider during setup or switch at any time from the settings panel.

Yes. {{ app_name }} is designed for self-hosted deployment via Docker Compose. You maintain full control over your data, assessment results, and infrastructure. No data ever leaves your environment unless you configure external AI providers.

Reports are auto-generated as PDF documents with CVSS-scored findings, evidence screenshots, remediation guidance, and executive summaries. Each finding includes detailed reproduction steps and is mapped to relevant CVE identifiers when applicable.

The platform ships with 25+ pre-configured tools including Nmap, Nuclei, Nikto, SQLMap, Feroxbuster, Hydra, and more. New tools can be added by dropping a JSON plugin file into the plugins directory — no code changes required.
