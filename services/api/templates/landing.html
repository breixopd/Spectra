Findings you can hand to an auditor

Scanners produce noise. {{ app_name }} separates confirmed exploitation from speculation with a verification pipeline built into every mission.

  • Evidence bundles Each finding stores the command, raw output, and hashed artifacts.
  • Proof status Verified, plausible, or rejected. The verifier re-runs exploits in a sandbox before marking anything confirmed.
  • Consensus gates High-risk decisions pass k-threshold voting across multiple model instances. Corrupted output gets discarded, not reported.

A full assessment pipeline, not a scanner wrapper

Specialist agents coordinate the same workflow a human team runs, from scoping to the final report.

Multi-agent missions

Twelve specialist agents split the work: scoping, reconnaissance, tool selection, exploitation, verification, and reporting. Plans adapt mid-mission when findings change the picture, without aborting the run.

Sandboxed execution

Every tool runs in an isolated container with scope limits, resource tiers, and automatic teardown. Nothing touches the host.

Methodology-driven

PTES, OWASP, and NIST phase models load from YAML. Run them advisory or strict.

Live attack graph

Frequently asked questions

What can {{ app_name }} test?

Network reconnaissance, vulnerability scanning, web application testing, service enumeration, and exploitation. Every tool runs in an isolated sandbox, and you control the scope and rules of engagement for each mission.

Is automated exploitation safe to run?

Tools execute in isolated containers with strict target boundaries. A safety supervisor blocks dangerous commands, and you can require human approval for high-risk actions. We recommend starting against staging environments.

Can I bring my own LLM API key?

Yes, on plans that include BYOK. {{ app_name }} routes inference through a provider-agnostic gateway, so OpenAI-compatible providers work out of the box.

How are reports generated?

Reports assemble automatically from verified findings: CVSS scores, evidence artifacts, reproduction steps, remediation guidance, and an executive summary. Export as PDF, JSON, or Markdown.

Which security tools are included?

{{ tools_count }} tool plugins ship today, including Nmap, Nuclei, SQLMap, Metasploit, Hydra, and Feroxbuster. The plugin system is JSON-defined, so new tools can be added without touching platform code.
