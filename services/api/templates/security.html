Security

How we protect your data and our commitment to security.

Infrastructure Security

  • All data encrypted in transit (TLS 1.3) and at rest (AES-256)
  • Tool execution isolated in sandboxed Docker containers
  • Network-isolated assessment environments with strict egress controls
  • Automated vulnerability scanning of platform dependencies
  • Regular security audits and penetration testing

Authentication & Access

  • Multi-factor authentication (TOTP)
  • Bcrypt password hashing with per-user salts
  • JWT tokens with short expiry and refresh rotation
  • Progressive account lockout on failed attempts
  • Role-based access control (Admin, Staff, User)
  • Comprehensive audit logging of all security events

Data Protection

  • Findings and assessment data stored with per-tenant isolation
  • Automatic data retention policies and secure deletion
  • Bring Your Own Key (BYOK) support for AI providers
  • No assessment data shared with third parties

Responsible Disclosure

We take security vulnerabilities seriously. If you discover a security issue, please report it responsibly.

Email: security@spectra.dev

Please include a detailed description, steps to reproduce, and potential impact.

We aim to acknowledge reports within 24 hours and provide a fix within 72 hours for critical issues.
